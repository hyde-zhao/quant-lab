- `BacktraderResult` / optional backend result / unavailable result 的 typed schema。
- backend selector 默认 `lightweight`，`backtrader` 只由显式参数启用。
- 未安装 Backtrader 返回 `backend_unavailable`，默认轻量主路径和默认 pytest 不受影响。

- Backtrader 只负责调仓、成交、成本、仓位、净值和风险分析。
- benchmark `unavailable` / `required_missing` 只报告对照缺失或 metadata，不触发 fetch/backfill/write。
